This repository includes Power BI project (Dashboards - Reports) leveraging data modeling, DAX measures, PB Queries and Visualizations for data analysis. 

1. Sales & Marketing Project v1

The data  that used in this project has been pushed from SSMS to PowerBI and comes from the 2nd Project from the "SQL-Projects-for-Data-Analysis" repository.
The initial idea was to push the silver layer to PB for some last data transformations, proceed to data modelling and create some DAX Measures.
It is important to be mentioned that in this Project, numbers might be sligthly different due to the fact that we decided to not include NULL customer_ids or gross_revenue in our analysis.
We wanted to keep our data as clean as possible and PowerBI features allows us to be more flexible with data.


This project is an end-to-end Business Intelligence solution built in Power BI, focused on analyzing e-commerce performance across sales, products, customers, and marketing campaigns. 
It combines data transformation (Power Query), star schema data modeling, and DAX measures to deliver interactive dashboards that provide insights into revenue trends, transaction behavior, and campaign effectiveness.
The report includes multiple analytical views such as Overview, Product, Campaign, and Transaction Analysis, supported by features like a custom date table, dedicated measures table, KPI calculations (Revenue, Transactions, Customers, AOV), and dynamic filtering. 
The goal is to transform raw data into actionable insights that support data-driven decision-making and highlight key business drivers.
